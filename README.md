
# Eniola's stuffs

## Usage

### 1. Clone the repo
```
git clone https://github.com/johnoseni1/Eniola-s-vesting-stacking-contract-UI.git
```

### 2. Install the dependencies
```
npm install
```

### 3. Start the web server
```
npm start
```

### 4. Ready!
Go to `http://localhost:3000` and interact with the contract!
